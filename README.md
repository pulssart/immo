# Simulateur de capacité d'achat immobilier

Simulateur interactif pour calculer votre capacité d'achat immobilier en fonction de vos revenus et des paramètres du crédit.

## Fonctionnalités

- Calcul automatique de la capacité d'achat
- Ajustement en temps réel des paramètres
- Affichage du reste à vivre avec indicateur de confort
- Interface simple et intuitive

## Déploiement

Ce projet est prêt à être déployé sur Netlify. Il suffit de connecter le dépôt GitHub à Netlify et le site sera automatiquement déployé.
